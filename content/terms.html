DECLARACIÓN DE PRIVACIDAD

La OIM está firmemente comprometida con la protección de datos relativa a los datos personales de los usuarios del portalwww.hilandocomunidades.com


Navegación anónima

Los usuarios pueden navegar los sitios web sin revelar sus datos personales.


Supervisión del tráfico del sitio web

La OIM supervisa las tendencias generales en el uso de sus sitios web.

Asimismo, utiliza la información para generar estadísticas y medir la actividad de los sitios web a fin de mejorar su utilidad.


Cookies

La supervisión del tráfico de los sitios web comprende el uso de cookies, pequeñas secuencias de código que se almacenan en el disco duro del ordenador del usuario. Cuando el usuario visita los sitios web, a efectos de Google Analytics, se recopilan datos como: Datos de registro (dirección IP, hora, fecha, sitio de referencia, navegador); Datos de seguimiento (cookies de una o varias sesiones); Datos de ubicación o GPS; y Configuración o preferencias del dispositivo (seguridad, uso compartido, estado).

El usuario tiene derecho a rechazar el uso de cookies siempre que su navegador lo permita, aunque, en tal caso, es posible que no pueda utilizar determinadas funciones de los sitios web y que necesite reintroducir información con mayor frecuencia para utilizar ciertos servicios de los sitios web.


Datos personales

La OIM no recopila datos personales de usuarios particulares, tales como el nombre, la dirección de correo, la dirección de correo electrónico, ni datos de otra índole.  

 

Enlaces

El usuario puede encontrar enlaces a sitios web de terceros en el sitio web de la OIM y otras páginas web relacionadas. Nótese que la OIM no es responsable de las políticas de privacidad o los contenidos de los sitios web de terceros.
